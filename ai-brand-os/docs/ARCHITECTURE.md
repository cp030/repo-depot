# Architecture

AI Brand OS is a Next.js 15 App Router application with the following structure:

## Route Groups

- `(admin)` — Internal dashboard for managing brand projects
- `(client)` — Client-facing intake and review flows

## Key Libraries

- **Supabase** — Database and auth
- **Anthropic SDK** — AI agent orchestration
- **Zustand** — Client-side state management
- **Framer Motion** — Animations
